作品名: プロフ帳管理アプリ

使用言語: JavaScript / HTML / CSS

概要: 管理人が画像を投稿し、閲覧者が各投稿に自分用タグや一言メモを追加できるプロフィール型画像アーカイブ。

工夫: 人気タグ・類似タグ提案、名前とタグの完全一致 / 部分一致検索、カテゴリ絞り込み、並び替えを実装。
